# Currency Converter (Python)

A command-line currency converter using ExchangeRate-API.

## Features
- Convert any currency to any other currency using live exchange rates with the use an API
- Uses USD as a bridge for conversions (Any --> USD --> Any)

## Requirements
- Python 
- requests library

## Setup
1. Get a free API key from [ExchangeRate-API](https://www.exchangerate-api.com/)
2. Open `currency_convertor.py` and replace `"Enter your own API Key here"` with your key
3. Install required python libraries
